Comparing Color Scales

When you select a color, two color scales will be generated for it:

  1. A color scale derived from the original color's RGB values by increment and decrement.
  2. A color scale that works by matching each color in the color scale to a desired relative luminance value.

You can show or hide either of the color scales or show both side by side for comparison.

This project was done to verify that luminance derived color scale is more reliable. Click on any color to copy its hexcode. Additionally, you can open the browser's console to view more details about the selected colors and the generated color scales.
